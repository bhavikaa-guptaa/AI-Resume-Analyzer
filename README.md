# AI Resume Analyzer

## Problem Statement
Recruiters often screen resumes manually, which is time-consuming and inefficient.
This project automates resume screening using NLP-based skill matching.

## Solution
The system compares a candidateâ€™s resume with a job description and calculates
a skill match percentage while highlighting missing skills.

## Tech Stack
- Python
- Streamlit
- Natural Language Processing (Keyword Matching)

## Features
- Resume vs Job Description comparison
- Skill match percentage
- Missing skill identification
- Simple web interface

## How to Run
1. Install dependencies  
   `pip install -r requirements.txt`

2. Run the app  
   `streamlit run app.py`
